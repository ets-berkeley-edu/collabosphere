Notice to Safari browser users

Open the Safari browser preferences

Change the cookies preferences

Begin using the SuiteC tools

SuiteC needs to set its cookies in the Safari web browser before the SuiteC tools can be used. (Note: other browsers do not require these permissions.)
Under the Safari browser menu bar, select the "Preferences..." option.
Select "Privacy" and choose "Always Allow" in the "Cookies and website data" section.
You're all set to start exploring the Asset Library, Engagement Index, and Whiteboards. Optional: return your Privacy preferences back to their original settings after using the SuiteC tools.

Launch failure

SuiteC was unable to launch the tool you requested. You may need to adjust Safari's settings on your device to allow SuiteC to set cookies.
Under your device's Settings app, select "Safari." Under "Privacy & Security," select "Block Cookies" and change the selected value to "Always Allow." You should now be able to launch the tool. Once the tool has launched, you can return your Block Cookies setting to its original value.
