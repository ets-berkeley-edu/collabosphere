Begin using the SuiteC tools

SuiteC needs to set its cookies in the Safari web browser before the SuiteC tools can be used. (Note: other browsers do not require these permissions.)
Under the Safari browser menu bar, select the "Preferences..." option.
Select "Privacy" and choose "Always Allow" in the "Cookies and website data" section.
You're all set to start exploring the Asset Library, Engagement Index, and Whiteboards. Optional: return your Privacy preferences back to their original settings after using the SuiteC tools.
